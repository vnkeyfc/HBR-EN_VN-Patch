# BepInEX và XUnity.AutoTranslator cho Heaven Burns Red EN

## Cách cài đặt (tạm thời)
1. Tải gói <a href="https://mega.nz/file/kcA0zAwa#lVU2b16_JSx5n0rEQfBAl_MAsEWKRgFVBy06yVKcwXc" >patch</a> của game (Có sẵn tùy chỉnh BepInEX và XUnity.AutoTranslator)
2. Giải nén "BepInEx-to-HBREN.rar" và sao chép toàn bộ nội dung bên trong folder "BepInEx-to-HBREN" vào thư mục chứa game (...\YostarGames\HeavenBurnsRed)
3. Tải folder Translations ở Repo thành một file zip, giải nén và sao chép folder Translations vào folder BepInEX.
4. Khởi động game (Hãy đợi một lúc cho BepInEX thiết lập khi lần đầu khởi động)
5. (Optional) Nếu muốn dịch tự động, nhấn tổ hợp Alt + 0 để mở bảng UI của XUnity.Autotranslator và chọn translator GoogleTranslateV2.

## Trường hợp có thể xảy ra
* Nếu font hiển thị trong game xuất hiện ký tự ô vuông, khởi động lại game sẽ khắc phục lỗi này.
